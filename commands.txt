Compilation commands to run in cosmolattice/build:

1.) No paralellisation:
rm CMakeCache.txt && cmake -DMODEL=mexhat -DCMAKE_EXPORT_COMPILE_COMMANDS=ON ../ && make cosmolattice

2.) Paralellisation in one direction:
rm CMakeCache.txt && cmake -DMPI=ON -DMODEL=mexhat -DCMAKE_EXPORT_COMPILE_COMMANDS=ON ../ && make cosmolattice

3.) Paralellisation in two directions:
rm CMakeCache.txt && cmake -DMPI=ON -DPFFT=ON -DMODEL=mexhat -DCMAKE_EXPORT_COMPILE_COMMANDS=ON ../ && make cosmolattice

Sometimes needed extra flag with custom FFTW3 and PFFT installation paths, e.g.:
-DCMAKE_PREFIX_PATH="/home/will/cosmolattice/dependencies/MyFFTW3;/home/will/cosmolattice/dependencies/MyPFFT"


Run commands:

1.) No paralellisation:
./mexhat input=../src/models/parameter-files/mexhat.in

2.) Paralellisation in one direction:
mpirun -n ... mexhat input=../src/models/parameter-files/mexhat.in

2.) Paralellisation in two directions:
mpirun -n ... mexhat input=../src/models/parameter-files/mexhat.in
